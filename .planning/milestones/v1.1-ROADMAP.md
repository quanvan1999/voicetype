# Roadmap: VoiceType v1.1 Smart Dictation

## Overview

v1.1 adds two enhancement features to the shipped v1.0 foundation: auto-translate (speak Vietnamese, get English text) and auto-segment (hands-free dictation with automatic text insertion on silence). Both features maintain 100% offline operation using WhisperKit's native translate task and SileroVAD neural voice detection.

## Phase Details

### Phase 5: Auto-Translate
**Goal**: User can dictate in Vietnamese and receive English text output
**Depends on**: Phase 4 (v1.0 complete)
**Requirements**: TRANS-01, TRANS-02, TRANS-03, TRANS-04

**Success Criteria** (what must be TRUE):
1. User can toggle translate mode on/off in Settings
2. When translate mode is on, Vietnamese speech produces English text at cursor
3. Translation works without internet connection
4. Translation uses single inference (no perceptible delay increase from v1.0)

**Plans:** 1 plan

Plans:
- [x] 05-01-PLAN.md — Implement translate toggle, WhisperKit .translate task, history tracking

### Phase 6: Auto-Segment
**Goal**: User can dictate continuously without manual hotkey release
**Depends on**: Phase 5
**Requirements**: SEG-01, SEG-02, SEG-03, SEG-04, SEG-05, SEG-06, SEG-07

**Success Criteria** (what must be TRUE):
1. User can enable toggle mode that auto-inserts text when silence is detected
2. User can configure silence threshold (1-5 seconds) in Settings
3. Visual feedback shows when segment boundary is detected
4. User can continue speaking while previous segment is being transcribed
5. Multiple pending segments are queued and processed in order

**Plans:** 5 plans (3 original + 2 gap closure)

Plans:
- [x] 06-01-PLAN.md — Core infrastructure: VADService, CircularAudioBuffer, TranscriptionQueue (Wave 1)
- [x] 06-02-PLAN.md — Integration: VAD in AudioRecorder, continuous mode in TranscriptionService (Wave 2)
- [x] 06-03-PLAN.md — UI: Settings for auto-segment, visual indicators for queue depth (Wave 3)
- [x] 06-04-PLAN.md — Gap closure: Fix text insertion and status transition bugs (UAT fixes 3, 4, 5, 6)
- [x] 06-05-PLAN.md — Gap closure: Fix hallucination, UI visibility, mode switch (UAT fixes 7, 8, 9, 10)
